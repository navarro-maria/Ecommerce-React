# App Talento Tech - E-commerce

Tienda online desarrollada con React.

## Tecnologías

React, Vite, Firebase (Auth + Firestore), React Router, CSS Modules.

## Instalación

npm install
npm run dev

## Funcionalidades

- Catálogo de productos con detalle
- Carrito de compras (Context API)
- Autenticación de usuarios (login/registro)
- CRUD de productos y cupones (admin)
- Diseño responsivo
